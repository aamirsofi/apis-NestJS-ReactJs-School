## Recommended: Shadcn/ui Inspired Theme

### Color Palette
```css
Primary: Indigo/Slate (professional, trustworthy)
Secondary: Purple/Pink (modern, engaging)
Accent: Blue/Cyan (highlights, CTAs)
Success: Green
Warning: Amber
Error: Red
Neutral: Slate/Gray
```

### Design Principles
1. **Clean & Minimal** - Less visual clutter
2. **Consistent Spacing** - 4px/8px grid system
3. **Subtle Shadows** - Depth without heaviness
4. **Modern Typography** - Clear hierarchy
5. **Accessible Colors** - WCAG AA compliant
6. **Smooth Animations** - Subtle, purposeful

### Key Components Style
- Cards: Subtle borders, soft shadows
- Buttons: Clear hierarchy, rounded corners
- Forms: Clean inputs, helpful labels
- Tables: Zebra striping, hover states
- Sidebar: Clean, organized navigation
